# KPI Builder untuk UMKM

Prototype frontend statis untuk membantu owner UMKM membaca KPI secara terstruktur, melihat hubungan sebab-akibat antar KPI, dan memahami prioritas tindakan bisnis.

## Halaman Prototype

- `index.html` — Landing page / company profile
- `dashboard.html` — Main KPI dashboard
- `kpi-tree.html` — KPI Tree view
- `kpi-detail.html` — KPI detail view
- `builder.html` — KPI builder prototype
- `settings.html` — Settings prototype

## Stack

- Native HTML
- Tailwind CSS Play CDN
- Static dummy data
- Tanpa backend, database, auth, API, atau framework JavaScript

## Dummy Context

- Business name: `UMKM Demo`
- Industry: `Retail & F&B`
- Period: `Juni 2026`
- Main objective: `Meningkatkan Profitabilitas Bisnis`

## Catatan

- Prototype ini mengikuti dokumen SOT di folder `docs/`.
- Semua halaman dirancang untuk demo, validasi UI, dan presentasi awal.
- Belum ada fungsi simpan, export nyata, login, atau integrasi data.
